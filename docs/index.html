不确定自己的架构？运行 uname -m： x86_64 → amd64 aarch64 → arm64

三步搞定

  1. 下载：按上方按钮选择对应发行版和架构的安装包。
  2. 安装：使用 apt 或 dnf 安装（上方命令已为你准备好）。 安装时会自动为 chrome-sandbox 设置 setuid 位，这是 Linux 启动应用所必需的。
  3. 启动：从应用菜单启动，或在终端运行 minimax-code。

故障排查

应用无法启动，或提示 chrome-sandbox 错误

安装包会自动给内置的 chrome-sandbox 设置 setuid 位（chmod 4755）。 如果你手动解包了文件，请运行： 

sudo chmod 4755 /opt/minimax-code/chrome-sandbox
GNOME 下没有托盘菜单

请安装 GNOME Shell 扩展 AppIndicator and KStatusNotifierItem Support。

Wayland 下任务栏图标显示为默认图标

已通过 package.json#name 修复。如果仍出现，请升级到最新版本。

如何升级？

直接安装新版本安装包即可，包管理器会自动替换旧版本，无需先卸载。

如何卸载？ 
sudo apt remove minimax-code   # Debian/Ubuntu
sudo dnf remove minimax-code   # Fedora/RHEL
arm64 支持情况

arm64 版本为实验性构建。如无必要，请优先使用 x86_64 版本。
